## Built With

- ReactJS
- CSS
- Eslint, Stylelint

## Getting Started

To get a local copy up and running clone the repo and run

`npm install`

### Install

to build a the project into `./build` folder run

`npm run build`

### Usage

to serve the application on `localhost` run

`npm start`
